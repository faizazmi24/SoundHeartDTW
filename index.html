03

Cara Kerja DTW

LANGKAH 01
Bangun matriks cost
Hitung jarak absolut |xi − yj| untuk setiap pasang titik dari dua sinyal. Hasilnya adalah matriks n×m.
LANGKAH 02
Isi accumulated cost
Gunakan dynamic programming untuk mengisi matriks secara kumulatif. Setiap sel mengambil nilai minimum dari tiga tetangganya (atas, kiri, diagonal).
LANGKAH 03
Traceback warping path
Lacak jalur optimal dari pojok kanan-bawah ke kiri-atas. Panjang path ini mencerminkan "seberapa banyak peregangan" dibutuhkan untuk menyelaraskan kedua sinyal.
