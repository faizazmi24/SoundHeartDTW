04

Cara Kerja DTW

LANGKAH 01
Bangun matriks cost
Hitung jarak absolut |xi − yj| untuk setiap pasang titik dari dua sinyal. Hasilnya matriks n×m.
LANGKAH 02
Isi accumulated cost
Dynamic programming mengisi matriks secara kumulatif. Tiap sel mengambil nilai minimum dari tiga tetangganya (atas, kiri, diagonal).
LANGKAH 03
Traceback warping path
Lacak jalur optimal dari pojok kanan-bawah ke kiri-atas. Nilai akhir = jarak DTW antara dua sinyal.
